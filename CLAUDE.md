# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a Spring Boot 4.0 application built with Kotlin and Gradle. The project uses Kotlin 2.2.0, JVM toolchain 21, and follows Spring Boot best practices with a multi-module structure.

## Instruction Files

Detailed guidance is split into topic-specific files under `.claude/`:

- `build.md` — build commands and Gradle wrapper usage
- `testing.md` — test commands and testing strategy
- `development.md` — code quality, naming conventions, and code structure
- `commits.md` — conventional commit format and examples
